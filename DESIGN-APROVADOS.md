# Layouts aprovados (22/03/2026)

Este documento registra os layouts aprovados recentemente para evitar perda de contexto durante iterações futuras no arquivo `app.pen`.

## Logo principal — Node `vwhJ6`

- Versão final baseada na referência `Zymrt`, mantendo o mascote jovem, sem barba, com óculos, chapéu de chef branco simples, camiseta vermelha e luvas brancas.
- O mascote segura um pão redondo dourado dentro de uma forma preta, com corpo enquadrado dentro de um anel duplo laranja com leve profundidade 3D.
- A faixa inferior é laranja vibrante com tipografia branca em caixa alta lendo `LANCHE DE PAI`.
- Fundo branco limpo, com render semi-3D e sombreado suave, garantindo coerência com o app atual.
- Em caso de novos prompts AI, manter estes atributos para preservar a identidade aprovada.

## Ícone Google — Node `lbYo9`

- Botão de login utiliza o ícone oficial "G" multicolorido dentro da elipse `lbYo9`.
- Preenchimento atual é uma imagem vetorial AI transparente com as quatro cores da marca.
- Manter proporção 20×20 px; qualquer troca deve preservar borda nula e contraste suficiente com o fundo claro.

## Versão mascote para a Home — Node `sIyoJ`

- Ícone compacto para uso na tela inicial do app, mantendo o mascote aprovado.
- Jovem pai limpo (sem barba) com óculos, chapéu de chef branco, camiseta vermelha e luvas claras.
- Segura pão redondo dourado em forma preta sobre um disco laranja com fundo bege, criando leitura rápida mesmo em 120 px.
- Usar como referência visual quando outras áreas precisarem de variação reduzida da marca principal.
